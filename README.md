# goodreads-list-scraper
Scrapes GoodReads List then output in table form

Pre-requisites:
* pip3 install simple_term_menu
* pip3 install rich
* pip3 install beautifulsoup4

Run Command:
* python3 main.py

As of 08/20/2021
* Currently works on the first page of the list

Roadmap:
