# goodreads-list-scraper
Scrapes GoodReads List then output in table form

Pre-requisites:
- beautifulsoup4
- rich
- simple_term_menu
- requests
```
pip3 install -r requirements.txt
```

Run Command:
```
python3 main.py
```
